Anime Data Dashboard

This dashboard provides information about popular animes. The user can filter animes by genre using the chord chart in the middle. The three other charts on the page update depending
on the genres selected on the chord chart. Each chart shows various attributes and features of those genres. The user can hover over the charts to see a tooltip displaying the data.

How to Use Chord Chart
Single Select: {LEFT CLICK}
Multi Select: {SHIFT} + {LEFT CLICK}
To un-select indivual use {SHIFT} + {LEFT CLICK} on a selectd node
